### 🚀 How to Test

1. **Start Backend:**
   ```bash
   cd ecom_backend
   python manage.py runserver
   ```

2. **Start Frontend:**
   ```bash
   npm start
   ```

3. **Test Complete Flow:**
   - Browse products at `http://localhost:3000`
   - Register/login with real authentication
   - Add items to cart and wishlist
   - Test all CRUD operations
